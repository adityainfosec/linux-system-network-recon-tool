# Linux System Network Recon Tool

## Description

A Python-based Linux reconnaissance tool that gathers system and network information for learning and security assessment purposes.

## Features

- Hostname Detection
- Local IP Address Detection
- MAC Address Detection
- Operating System Information
- Kernel Version Detection
- Architecture Information
- Network Interface Enumeration
- LAN Device Discovery
- JSON Report Generation

## Requirements

- Python 3
- Linux Operating System
- ip command available in PATH

## Usage

Run:

```bash
python3 recon_tool.py
```

## Sample Output

- Hostname
- IP Address
- MAC Address
- Network Interfaces
- Connected LAN Devices
